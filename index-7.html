ACTONBOX WORKOUTS

ACTONBOX

Local Setup: Download this index.html to a folder. Run a local web server (camera requires HTTPS or localhost, NOT file://):

cd /path/to/folder
python3 -m http.server 8000

Then open http://localhost:8000 in your browser.

Wi-Fi: Required on first run to fetch TF.js + MoveNet from the CDN (~5 MB). Browsers cache these, so re-runs work offline until cache clears.

Storage: Workouts save to this browser's localStorage, on this device only. Up to 50 sessions kept. Clearing browser data deletes them.

Privacy: Camera frames never leave your device. Only the open-source pose model is loaded over the network.
